## Description
ASP.NET Core Web API that allows renting cars. Has options of
- getting selected or all cars
- adding new cars
- deleting existing ones
- updating existing ones
- renting cars

## Tech Stack
- ASP.NET Core
- EF Core Sqlite
- Swagger

## Setup
1. git clone https://github.com/seweryndawi/CarRentalProject.git
2. cd CarRentalProject
3. dotnet run
/
Testing possible using
- .http file or
- swagger UI at http://localhost:5199/swagger
